`rustup update`:获取最新版本的 Rust

`cargo --version`:要检查您是否安装了 Rust 和 Cargo，可以在终端中运行

`cargo build`: 可以构建项目

`cargo run`: 可以运行项目

`cargo test`: 可以测试项目

`cargo doc`: 可以为项目构建文档

`cargo publish`: 可以将库发布到 crates.io。

## 创建新项目
`cargo new``cargo new hello-rust`

`Cargo.toml`:为 Rust 的清单文件。其中包含了项目的元数据和依赖库

`src/main.rs`: 为编写应用代码的地方
